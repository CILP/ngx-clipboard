# ngx-clipboard

 Angular2  directive for [clipboard.js](http://zenorocha.github.io/clipboard.js/) by [@zenorocha](https://twitter.com/zenorocha)

## Install

You can get it on npm.

```
npm install ngx-clipboard --save
```
